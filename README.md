## 시작하기

### 필수 조건

- Node.js 18.0.0 이상
- npm 또는 yarn

### 설치 방법

```bash
# 저장소 클론
git clone https://github.com/yourusername/ntb_members.git

# 프로젝트 디렉토리로 이동
cd ntb_members

# 의존성 설치
npm install
# 또는
yarn install
```

### 실행 방법

```bash
# 개발 서버 실행
npm run dev
# 또는
yarn dev
```
